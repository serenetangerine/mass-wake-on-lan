# mass-wake-on-lan
#Overview:#
A script that reads from a txt file to send wake on LAN calls using etherwake.
Default txt file is macaddrs.txt.

#Dependencies:#
- `etherwake` 
    - Debian Based (Ubuntu, raspiOS, etc...):
        - `sudo apt install etherwake`
    - Arch Linux
        - `sudo yum install etherwake`
